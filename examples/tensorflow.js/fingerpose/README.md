# Wechaty + TensorFlow.js FingerPose

## Requirements

1. Node.js v12+
1. Wechaty Token

## Steps

### 1. Build a Wechaty Bot

```sh
npm i wechaty @chatie/tsconfig
```

### 2. Test Handpose Model

```sh
npm i @tensorflow/tfjs-node @tensorflow-models/handpose
npm i canvas file-box
```
